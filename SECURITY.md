| Version | Supported          |
| ------- | ------------------ |
| 0.1     | :x:                |

## Reporting a Vulnerability

If you find a Security vulnerability, please create [a new issue](https://github.com/TheAcharya/MarkersExtractor/issues) on GitHub. A fix will be issued as soon as possible.
